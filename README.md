# 🛒 EliteCart – Modern E-Commerce Frontend

A modern, fully responsive e-commerce frontend built with Angular.

## ✨ Features

- 🏠 Premium home page
- 🔍 Product search, filter, sorting
- 📄 Product details page
- 🛒 Cart with real-time updates
- 💳 Checkout flow
- ❤️ Wishlist
- 🔐 Auth UI (login/register)
- 📦 Order system (frontend)
- ⚡ Loading skeletons
- 🔔 Toast notifications
- 📱 Fully responsive design

## 🛠 Tech Stack

- Angular (Standalone + Signals)
- TypeScript
- SCSS
- LocalStorage (state persistence)

## 🚀 Run locally

```bash
npm install
ng serve
